## 🚀 使用方法

### 访问网站
- 主页：打开 `index.html`（自动跳转到 `pages/index.html`）
- 建筑页面：`pages/buildings.html`
- 工具页面：`pages/tools.html`
- 关于页面：`pages/about.html`
- 详情页面：通过卡片点击进入

### 使用工具
```bash
# 管理系统
cd tools
python manage_system.py

# 上传工具
cd tools
python upload_tool.py
```
